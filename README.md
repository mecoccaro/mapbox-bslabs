# mapbox-bslabs

## Requisitos

* Instalar paquetes
```
npm install
```

* Agregar credenciales de mapbox a las variables de ambiente de angular ``/mapbox-angular/src/enviroments/enviroments.ts`` Crear la variable ``mapboxKey``

## Para ejecutar el programa

```
cd mapbox-angular
```
```
ng serve --open
```
